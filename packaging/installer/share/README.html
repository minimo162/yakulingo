セットアップ完了 / YakuLingo 使い方ガイド

翻訳を、仕事の流れのままに。

常駐で、コピー操作だけで翻訳。テキストもファイルも作業の流れを止めません。

操作は Ctrl+C を2回

同じウィンドウで素早く2回。

終了はスタートメニュー

UI を閉じても常駐。終了は「YakuLingo 終了」。

自動判定: 日本語/英語 テキスト/ファイル対応
  • テキスト翻訳
  • ファイル翻訳
  • 常駐 / 終了

01テキスト翻訳

翻訳したい文章を選び、Ctrl + C を素早く2回。

日本語 → 英語

標準 / 簡潔 / 最簡潔を同時に表示。

英語 → 日本語

翻訳＋解説をまとめて表示。

02ファイル翻訳

ファイルを選択して Ctrl + C を素早く2回。

  • 進捗はファイル翻訳パネルで確認。
Excel Word PowerPoint PDF TXT MSG
レイアウトは可能な限り維持。フォントは翻訳方向で自動調整。

03常駐 / 終了

  • UI を閉じても翻訳は常駐で継続。
  • PC 再起動後は自動で常駐起動します。
  • UI を開く: スタートメニュー > YakuLingo。
  • 終了する: スタートメニュー > YakuLingo > 「YakuLingo 終了」。
  • アンインストール: スタートメニュー > YakuLingo > 「YakuLingo アンインストール」。
