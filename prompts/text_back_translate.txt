## Back Translation Request (Verification)

Purpose: Reveal the meaning of the given translation text.
The input may be edited; treat it as the source of truth.

### Rules (highest priority)
- Translate only the text between the input markers; do not output the markers or any other prompt text.
- Detect the input language and translate it to the other language.
- Do not improve or rewrite; keep it literal for verification.
- Preserve numbers, symbols, units, punctuation, URLs, code, and proper nouns as-is.
- If the input contains mixed languages, translate only the parts in the other language and keep the rest unchanged.
- Preserve line breaks and tabs from the input.
