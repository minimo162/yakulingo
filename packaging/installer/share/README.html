翻訳を、仕事の流れのままに。

YakuLingo は常駐型の翻訳アプリです。テキストもファイルも、作業を止めずに翻訳できます。

最初の一歩を始める 使い方の流れを見る
覚える操作はひとつだけ

同じウィンドウで Ctrl + C を素早く2回。

テキストもファイルも対応

Excel / Word / PowerPoint / PDF / TXT / MSG を翻訳できます。

終了は明示的に

UI を閉じても常駐。終了はスタートメニューの「YakuLingo 終了」から。

01最初の一歩

1
Copilot にサインイン

初回のみ、M365 Copilot のログインが必要です。案内に従ってログインしてください。

2
翻訳したい文章やファイルを選択

アプリを切り替えず、いつもの作業画面のまま選択します。

3
Ctrl + C を素早く2回

YakuLingo が翻訳を開始し、UI に結果が表示されます。

自動判定で迷わない。 日本語なら英訳、英語なら和訳。方向は自動で切り替わります。

02使い方の流れ

1. コピーで起動

いつもの操作で翻訳を呼び出せるので、作業のテンポが崩れません。

2. 翻訳結果を確認

日本語→英語は 3 スタイル、英語→日本語は解説付きで表示されます。

3. 必要なら再翻訳

文調や用途に合わせて再翻訳や戻し訳で調整できます。

4. 作業へすぐ戻る

翻訳後は UI を閉じても常駐。次の翻訳にすぐ移れます。

03テキスト翻訳

基本操作

  1. 翻訳したい文章を選択。
  2. 同じ画面で Ctrl + C を素早く2回。
  3. UI に翻訳結果が表示されます。
日本語 → 英語

標準 / 簡潔 / 最簡潔の 3 スタイルを同時に表示します。

英語 → 日本語

翻訳＋解説をまとめて表示。背景理解まで一気に確認できます。

戻し訳で精度確認

翻訳結果をワンクリックで逆方向に翻訳できます。

04ファイル翻訳

開始方法

  • エクスプローラーでファイルを選択。
  • Ctrl + C を素早く2回。
  • YakuLingo のファイル翻訳パネルで進行状況を確認。

対応ファイル

Excel Word PowerPoint PDF TXT MSG
レイアウトや書式は可能な限り保持して翻訳します。フォントは翻訳方向に合わせて自動調整されます。

05結果の受け取り

出力ファイル

翻訳済みファイル、対訳ファイル、用語集 CSV が一覧で表示されます。

保存先の確認

出力先フォルダーは UI から直接開けます。

履歴もローカル管理

翻訳履歴は端末内に保存され、再確認が簡単です。

06ビジネス運用にうれしいポイント

参考ファイルで表現を統一

用語集やスタイルガイドを添付して、チーム内の表記ルールを守れます。

自動言語判定

翻訳方向を考える手間なし。作業スピードを落としません。

常駐でいつでも翻訳

呼び出しはいつも同じ操作。新人メンバーでも迷いません。

ポイント: 翻訳が途切れる場合は Copilot のログイン状態を確認してください。

07常駐 / 終了

  • UI を閉じても翻訳は常駐で継続します。
  • UI を開く: スタートメニュー > YakuLingo。
  • 終了する: スタートメニュー > YakuLingo > 「YakuLingo 終了」。

08困ったとき

ショートカットが反応しない

同じウィンドウで Ctrl + C を素早く2回押してください。

翻訳が始まらない

Copilot にログイン済みか確認し、UI の再起動を試してください。

翻訳品質を上げたい

用語集・スタイルガイドを添付して表現のゆらぎを減らします。

09さっそく使ってみる

翻訳したいテキストやファイルを選び、Ctrl + C を素早く2回。YakuLingo がすぐに動き出します。
