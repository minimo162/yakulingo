Ctrl+Alt+J で翻訳。

Microsoft 365 Copilot を翻訳エンジンとして利用する、常駐型の翻訳支援ツールです。

テキストもファイルも Ctrl+Alt+J

テキスト/ファイルを選択して Ctrl+Alt+J を押すと翻訳を開始します。

英訳/和訳は自動判定

日本語 → 英語（3スタイル）/ それ以外 → 日本語（解説つき）。

初回のみ：Copilot にログイン

状態表示が「ログインが必要」の場合、画面上部の警告バーにある［Edgeを前面表示］をクリックして、Copilot にログインしてください。

  • テキスト翻訳
  • ファイル翻訳
  • 常駐 / 終了

01テキスト翻訳

文章を選択して Ctrl + Alt + J を実行してください。

日本語 → 英語

標準 / 簡潔 / 最簡潔を並列表示します。

英語 → 日本語

翻訳と解説を併記して表示します。

02ファイル翻訳

エクスプローラーでファイルを選択して Ctrl + Alt + J を実行してください（最大10件）。

  • ファイル翻訳画面の「翻訳するファイルをドロップ」にドラッグ＆ドロップ（またはクリックして選択）でも翻訳できます。
  • 進捗はファイル翻訳画面で確認できます。
Excel Word PowerPoint PDF TXT MSG
レイアウトは可能な限り維持し、フォントは翻訳方向で自動調整します（英訳: Arial / 和訳: MS Pゴシック）。

03常駐 / 終了

  • UI を閉じても常駐します（終了は「YakuLingo 終了」を使用）。
  • PC 再起動後も自動で常駐起動します。
  • 翻訳開始（UI表示）: Ctrl+Alt+J（テキスト/ファイルを自動判別）。
  • 常駐を起動: スタートメニュー > YakuLingo（常駐起動）。
  • 終了: スタートメニュー > YakuLingo 終了。
  • アンインストール: スタートメニュー > YakuLingo アンインストール。
翻訳中は Edge（Copilot）が背後で起動/最小化されることがあります。初回はログインが必要です。
YakuLingo - 常駐翻訳ツール（Microsoft 365 Copilot）。
