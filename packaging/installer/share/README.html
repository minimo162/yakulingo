YakuLingo / 使い方ガイド

翻訳を、いつでも即時に。

YakuLingo は常駐型の翻訳アプリです。普段の作業を止めずに、テキストもファイルもすぐ翻訳できます。

クイック起動 同じウィンドウで Ctrl + C を素早く2回。
ファイル翻訳 ファイルを選択して Ctrl + C を素早く2回。
終了方法 UIを閉じても常駐。終了はスタートメニューの「YakuLingo 終了」から。
  • 最初の一歩
  • テキスト翻訳
  • ファイル翻訳
  • 常駐/終了
  • 困ったとき

最初の一歩

1. セットアップ完了 setup.ps1 実行後、アプリは自動で常駐します。
2. Copilot にログイン 初回は M365 Copilot のログインが必要です（案内が表示されたらログイン）。
3. 翻訳開始 テキストやファイルを選択して Ctrl + C を素早く2回。
翻訳方向は自動判定です。日本語なら英訳、英語なら和訳が標準で表示されます。

テキスト翻訳

  1. 翻訳したい文章を選択します。
  2. 同じウィンドウで Ctrl + C を素早く2回。
  3. UIに翻訳結果が表示されます。必要に応じてコピーします。
日本語から英語は「標準/簡潔/最簡潔」を同時に表示。英語から日本語は解説付きで表示されます。

ファイル翻訳

開始方法

  • エクスプローラーでファイルを選択し、Ctrl + C を素早く2回。

翻訳結果

  • 進捗と完了ステータスが UI に表示されます。
  • 完了後、翻訳済みファイルや対訳ファイルが一覧表示されます。

常駐/終了

  • UIを閉じても常駐は継続します。
  • UIを開く: スタートメニュー > YakuLingo。
  • 終了する: スタートメニュー > YakuLingo >「YakuLingo 終了」。
常駐起動により、いつでもショートカットで翻訳できます。

困ったとき

翻訳が始まらない Copilot のログインが完了しているか確認してください。
UIが表示されない スタートメニューから YakuLingo を起動して確認してください。
ファイル翻訳で失敗する ファイルが開かれていないか、サイズが大きすぎないか確認してください。
インストール先: %LOCALAPPDATA%\YakuLingo
