============================================================
YakuLingo 配布方法（管理者向け）
============================================================

【配布フォルダの構成】

共有フォルダに以下のファイルを配置してください：

  \\server\share\YakuLingo\
    ├── インストール.bat      ← ユーザーがダブルクリック
    ├── install.ps1           ← インストール処理本体
    ├── YakuLingo_YYYYMMDD.zip ← 配布パッケージ
    └── README_配布方法.txt    ← このファイル

------------------------------------------------------------

【ユーザー向け手順】

ユーザーには以下のように案内してください：

  1. 共有フォルダ \\server\share\YakuLingo を開く
  2. 「インストール.bat」をダブルクリック
  3. 「Y」を入力してEnter
  4. 完了後、デスクトップの「YakuLingo」から起動

※ ZIPのダウンロードや展開は自動で行われます

------------------------------------------------------------

【配布パッケージの更新方法】

1. 開発環境で make_distribution.bat を実行
2. 生成された YakuLingo_YYYYMMDD.zip を共有フォルダにコピー
3. 古いZIPファイルは削除してください
   （install.ps1は最新のZIPを自動検出します）

------------------------------------------------------------

【動作要件】

- Windows 10/11
- PowerShell 5.1以上（標準インストール済み）
- 共有フォルダへの読み取り権限

------------------------------------------------------------

【トラブルシューティング】

エラー: 「実行ポリシーによりスクリプトが実行できません」
→ インストール.batから実行すれば自動的にBypassされます
   直接install.ps1を実行しないでください

エラー: 「ZIPファイルが見つかりません」
→ 共有フォルダに YakuLingo*.zip があるか確認してください

エラー: 「アクセスが拒否されました」
→ ユーザーに共有フォルダの読み取り権限があるか確認
