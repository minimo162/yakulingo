YakuLingo / 使い方ガイド

翻訳を、作業の流れのままに。

YakuLingo は常駐型の翻訳アプリです。テキストもファイルも、作業を止めずに翻訳できます。

覚える操作は1つだけ 同じウィンドウで Ctrl + C を素早く2回。
テキストもファイルも対応 Excel / Word / PowerPoint / PDF / TXT / MSG。
終了は明示的に UIを閉じても常駐。終了はスタートメニューの「YakuLingo 終了」から。
  • 最初の一歩
  • テキスト翻訳
  • ファイル翻訳
  • 結果の受け取り
  • 常駐/終了
  • 困ったとき

最初の一歩

1. セットアップ完了 インストール後、YakuLingo は自動で常駐します。
2. Copilot にログイン 初回は M365 Copilot のログインが必要です（案内が表示されたらログイン）。
3. 翻訳を開始 対象を選択して Ctrl + C を素早く2回。
自動判定 日本語なら英訳、英語なら和訳が自動で表示されます。

テキスト翻訳

  1. 翻訳したい文章を選択します。
  2. 同じウィンドウで Ctrl + C を素早く2回。
  3. UIに翻訳結果が表示されます。必要に応じてコピーします。
日本語 → 英語は「標準 / 簡潔 / 最簡潔」を同時に表示。英語 → 日本語は解説付きで表示されます。

ファイル翻訳

開始方法

  • エクスプローラーでファイルを選択し、Ctrl + C を素早く2回。

対応ファイル

  • Excel / Word / PowerPoint / PDF / TXT / MSG

結果の受け取り

進捗の確認 翻訳中は UI に進捗と状態が表示されます。
出力ファイル 完了後、翻訳済みファイルや対訳ファイルが一覧表示されます。
参照ファイル 用語集やスタイルガイドを添付すると翻訳が安定します。

常駐/終了

  • UIを閉じても常駐は継続します。
  • UIを開く: スタートメニュー > YakuLingo。
  • 終了する: スタートメニュー > YakuLingo > 「YakuLingo 終了」。
常駐起動により、いつでもショートカットで翻訳できます。

困ったとき

ショートカットが反応しない 必ず同じウィンドウで Ctrl + C を素早く2回押してください。
翻訳が始まらない Copilot のログインが完了しているか確認してください。
UIが表示されない スタートメニューから YakuLingo を起動して確認してください。
インストール先: %LOCALAPPDATA%\YakuLingo
