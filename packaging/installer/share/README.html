フォルダ構成

共有フォルダに以下を配置してください。

\\server\share\YakuLingo\
  setup.vbs              <-- 実行するファイル
  YakuLingo_YYYYMMDD.zip <-- 配布パッケージ
  README.html            <-- このガイド
  .scripts\              <-- 内部スクリプト（変更禁止）

ユーザー向け手順

  1. \\server\share\YakuLingo を開く。
  2. setup.vbs をダブルクリック。
  3. セットアップ完了まで待つ。
  4. YakuLingo はログオン時に常駐起動（セットアップ完了直後にも起動）。
  5. 翻訳の使い方:
    • テキスト: 同じウィンドウで Ctrl+C を短時間に2回 → UIに結果表示。
    • ファイル: エクスプローラーで同じウィンドウ内 Ctrl+C を短時間に2回 → Fileタブに結果表示。
    • ファイル（右クリック）: 対応ファイルを右クリック → 「YakuLingoで翻訳」（Windows 11 は「その他のオプション」内）。
  6. UIを開く: デスクトップ or スタートメニューのショートカット（UIを閉じても常駐は終了しません）。
  7. 終了: スタートメニュー > YakuLingo > 「YakuLingo 終了」。
注: インストール先は %LOCALAPPDATA%\YakuLingo（OneDrive同期外）。

更新手順

  1. 開発環境で以下を実行:
    • 必要に応じて install_deps.bat。
    • make_distribution.bat を実行。
  2. 新しい YakuLingo_YYYYMMDD.zip を共有フォルダへコピー。
  3. 古い ZIP は削除（setup.ps1 は最新ZIPを使用）。

要件

OS

Windows 10/11

PowerShell

PowerShell 5.1+（Windows標準）

権限

共有フォルダへの読み取り権限

トラブルシュート

スクリプト実行が無効

setup.vbs を実行してください（.scripts\setup.ps1 を直接実行しない）。VBS が実行ポリシーを調整します。

ZIP が見つからない

同じフォルダに YakuLingo*.zip があるか確認してください。

Access denied

共有フォルダの読み取り権限を確認してください。
