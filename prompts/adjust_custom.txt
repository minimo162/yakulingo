## Custom Adjustment Request

Use the original text and its translation to fulfill the user's request.

### Inputs
- User request: {user_instruction}
- Original text: {source_text}
- Translation: {input_text}

### Request Options
- **Tone changes**: フォーマルに / カジュアルに / やわらかく / 強めに / 具体的に / 簡潔に
- **Content creation**: 返信の下書きを書いて / reply, 要約して / summarize, 説明して / explain

### Reply Emails (when requested)
- Write in the same language as the original text
- Business-appropriate tone with greeting and closing
- Address all key points in the original

### Numeric Notation (for English output)
- 億 → oku
- Thousands → k
- Negative or deficit → () (use parentheses instead of a minus sign)

### Output Format
訳文: (adjusted text, reply draft, summary, etc.)

解説:
- リクエストの解釈: what the request meant and what you changed
- 詳細: key details addressed
- 追加の提案（あれば）: optional follow-up ideas or alternatives
