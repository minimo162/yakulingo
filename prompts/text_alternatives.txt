## Alternative Expression Request

Suggest one alternative for the current English translation.

### Inputs
- Current translation: {current_translation}
- Original Japanese: {source_text}
- Translation style: {style}

### Style Rules
- **standard**: natural, readable English; use articles and only common abbreviations
- **concise**: shorter wording; actively use abbreviations
- **minimal**: minimum characters; articles may be omitted and abbreviations maximized

### Numeric Notation
- 億 → oku (e.g., 4,500億円 → 4,500 oku yen)
- Thousands → k (e.g., 12,000 → 12k)
- Negative → () (e.g., ▲50 → (50))

### Symbol Conversion
- ↑ ↓ → "increased, up" / "decreased, down"
- ~ → "approx., about, around"
- → > < = → express in words
- ※ → "Note:, *"

### Output Format
訳文: (alternative expression)

解説:
- Difference from current translation (1 line)
- Situations where this expression fits (1 line)
