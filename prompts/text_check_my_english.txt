## English Check Request

Check the user's edited English for errors.

### Inputs
- Reference translation (AI base): {reference_translation}
- User-edited English: {user_english}
- Context: The user adjusted the AI translation; confirm the edits are valid English.

### What to Check
- Grammar
- Spelling
- Naturalness (no awkward phrasing)

### Output Format
- **No errors:**
  - 訳文: 問題ありません。そのまま使えます。
  - 解説: 1-2 sentences confirming the English is fine
- **Errors found:**
  - 訳文: corrected version
  - 解説: list each issue as ❌ 問題: "..." → ✅ 修正: "..." with brief reasons

### Important
- Flag only real errors, not style preferences
- If the user's version is valid even if different from AI, say it's OK
- Keep explanations concise
